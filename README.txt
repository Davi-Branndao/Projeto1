ESTRUTURA DE ARQUIVOS:
  index.html  →  O site completo (único arquivo necessário)

----------------------------------------------
OPÇÕES GRATUITAS DE HOSPEDAGEM:
----------------------------------------------

1. NETLIFY (recomendado — mais fácil)
   • Acesse: https://netlify.com
   • Crie uma conta gratuita
   • Arraste a pasta "hospedagem-enem" para o painel
   • Pronto! Seu site estará no ar em segundos.

2. VERCEL
   • Acesse: https://vercel.com
   • Crie uma conta gratuita
   • Clique em "Add New Project"
   • Faça upload da pasta ou conecte ao GitHub
   • Deploy automático.

3. GITHUB PAGES (gratuito)
   • Crie uma conta em https://github.com
   • Crie um repositório novo
   • Faça upload do arquivo index.html
   • Vá em Settings > Pages > selecione branch "main"
   • Seu site ficará em: https://seuusuario.github.io/nome-do-repo

4. TIINY.HOST (mais simples de todos)
   • Acesse: https://tiiny.host
   • Faça upload do arquivo index.html diretamente
   • Receba um link instantâneo (sem criar conta)

----------------------------------------------
DOMÍNIO PERSONALIZADO (opcional):
----------------------------------------------
Se quiser um domínio próprio (ex: www.meuenem.com.br):
  • Registre em: https://registro.br (domínios .com.br)
  • Ou em: https://namecheap.com (domínios internacionais)
  • Aponte o domínio para o serviço de hospedagem escolhido.

----------------------------------------------
